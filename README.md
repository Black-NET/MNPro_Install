# MNPro_Install

## Auto installation for Ubuntu MNPro

git clone  https://github.com/Black-NET/MNPro_Install.git MNPro_Install

cd MNPro_Install

chmod +x MNPro_Install.sh

./MNPro_Install.sh

=========================================================================

cd /usr/local/bin

./mnpro-cli getinfo

./mnpro-cli masternode status

./mnpro-cli mnsync status

./mnpro-cli stop

./mnprod
